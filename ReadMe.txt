Program Description: 

COVID-19 Speech agent that provides the latest stats for CoronaVirus.
It provides data for Countries, States, Counties as well as Worldwide.
The different types of stats are confirmed case, deaths and recovered.
Values for different time periods can also be requested.
Unity is the GUI used which interacts with Google DialogFlow to receive the right response for queries. Google DialogFlow gets the latest data from the coronoaviris-tracker-api (https://coronavirus-tracker-api.herokuapp.com).

Tools Used:

1) Google Dialogflow (https://dialogflow.com/)
2) Firebase (https://firebase.google.com/)
3) Unity (https://unity.com/)
4) macOS Mojave

Dependencies:

1) Unity software.
2) Internet connection to connect to DialogFlow.
3) coronoaviris-tracker-api should be live.

How to compile and Run:

Navigate to the folder: \COVID AGENT\Build\BuildWindows\CovidAgent
Run the "COVID AGENT.exe" file.

To open in developer mode and compile, Open the COVID AGENT folder in Unity for mac.

Download files from https://drive.google.com/drive/folders/1aeyPSTSMAqb1D5mNdzRatkrvwe4og9FT?usp=sharing and place them in COVID19-Speech-Agent/COVID\ AGENT/Assets/Avatar.
